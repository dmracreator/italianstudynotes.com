# Study Notes — Italian A2 Course

## Open the website

1. Unzip the folder.
2. Open `index.html` in Chrome, Edge, Firefox, or Safari.
3. No installation or web server is required.

## Folder structure

```text
study-notes/
├── index.html
├── README.md
└── assets/
    ├── css/
    │   └── styles.css
    └── js/
        ├── data.js
        └── app.js
```

## Included functions

- Twenty Italian A2 conversation units
- Expanded vocabulary lists for everyday conversations
- Useful spoken sentences and natural native-style expressions
- Dialogues and reading texts
- Speaking prompts
- Vocabulary tests
- Sentence-reply practice
- Mixed tests
- Automatic scoring
- Progress stored in the browser with `localStorage`
- Modern iOS-inspired interface with system fonts and neutral colours
- Responsive design for desktop and mobile

## Edit course content

All course words, sentences, dialogues, and test prompts are in:

`assets/js/data.js`

You can edit the text directly in any text editor.

## Reset progress

Open the **Progress** page and click **Reset progress**.
